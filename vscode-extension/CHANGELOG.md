# Changelog

所有版本的变更记录。

## [1.3.0] - 2025-12-11

### 🎉 开源发布
- GitHub 仓库公开
- 配置从 GitHub 动态获取
- 支持社区贡献

### ✨ 新增功能
- **清空项目配置**按钮 - 一键删除项目的所有 Copilot 配置
- **启动时自动刷新** - 从 GitHub 获取最新配置列表
- **Custom Agents 支持** - 混合使用 copilot-instructions.md 和 .github/agents/

### 🔄 操作流程优化
- 新流程：先选择项目 → 勾选配置 → 应用
- 工具栏按钮重新排序和优化图标

### 🛠️ 技术改进
- Agent 文件自动复制到 .github/agents/ 目录
- 改进的错误处理和日志输出
- 自动添加配置文件到 .gitignore

详见：[CHANGELOG_v1.3.0.md](docs/releases/CHANGELOG_v1.3.0.md)
